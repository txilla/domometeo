Description

Our project controls essential things at home (inside and outside).

It involves the monitoring of:

  • Temperature, Atmospheric Pressure and Humidity.
  • Precipitation.
  • Doors/Windows Open/Close.
  • Movements Around the House.
  • Air Quality.
  • Light Level.

In order to control all of that, it’s necessary to:

  • Collect Data From All the Sensors by two Arduinos
  • Send the data from the Arduinos to Raspberry Pi with radio antennas through MySensors.org protocol
  • When data arrives to Raspberry through MySensors a Java Controller analyzes and stock it into the database
  • Finally the data is shown in the web page by PHP requests to database
