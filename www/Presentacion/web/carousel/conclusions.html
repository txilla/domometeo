Conclusions

We have learnt a lot with this project:

  • We have used a number of things studied in class ( Java, PHP, Ajax, HTML, Css,Sockets,BBDD) but also new things like Arduino in C++ or JQUERY.

We have combined diferent areas in computer science:

  • We have learnt how to connect electronic components
  • Low level programming to interact with sensors
  • Process sensor data with Java and Sockets and insert into the database

Interesting future extensions:

  • Involve Telegram and notifications
  • Control of more things at home such as turning the alarm on in the presence of dangerous gases
